Linux常用命令全拼

  • pwd：打印工作目录打印当前目录显示出当前工作目录的绝对路径

  • ps：进程状态（进程状态，类似于windows的任务管理器）

    常用参数：-auxf

    ps -auxf显示进程状态

  • df：disk free其功能是显示磁盘可用空间数目信息及空间结点信息。换句话说，就是报告在任何安装的设备或目录中，还剩多少自由的空间。

  • du：磁盘使用情况

  • rpm：即RedHat包管理，是RedHat的发明之一

  • rmdir：删除目录（删除目录）

  • RM：删除（删除目录或文件）

  • cat：连接连锁

  • cat file1file2 >> file3把文件1和文件2的内容联合起来放到file3中

  • insmod：安装模块，载入模块

  • ln -s：link -soft创建一个软链接，相当于创建一个快捷方式

  • mkdir：Make Directory（创建目录）

  • 触摸：触摸

  • 男子：手动

  • su：Swith用户（切换用户）

  • cd：更改目录

  • ls：列出文件

  • ps：流程状态

  • mkdir：制作目录

  • rmdir：删除目录

  • mkfs：制作文件系统

  • fsck：文件系统检查

  • uname：Unix名称

  • lsmod：列出模块

  • mv：移动文件

  • rm：删除文件

  • cp：复制文件

  • ln：链接文件

  • fg：前景

  • bg：背景

  • chown：更改所有者

  • chgrp：改变组

  • chmod：改变模式

  • 卸载：卸载

  • dd：本来应根据其功能描述“转换副本”命名为“cc”，但“cc”已经被用以代表“CComplier”，所以命名为“dd”

  • tar：磁带存档（磁带档案）

  • ldd：列出动态依赖项

  • insmod：安装模块

  • rmmod：删除模块

  • lsmod：列表模块

  • 文件结尾的“rc”（如.bashrc，.xinitrc等）：资源配置

  • Knnxxx / Snnxxx（位于rcx.d目录下）：K（Kill）; S（服务）; nn（执行顺序号）; xxx（服务标识）

  • .a（扩展名a）：存档，静态库

  • .so（扩展名so）：共享对象，动态链接库

  • .o（扩展名o）：目标文件，C / C ++源文件的编译结果

  • RPM：红帽包管理器

  • dpkg：Debian包管理器

  • apt：高级包工具（Debian或基于Debian的发行版中提供）
